# Witcher Adventure Game (JavaFX & Text UI)

Welcome to *Witcher Adventure*, an immersive text-based and GUI-enhanced adventure game built in Java using the JavaFX framework. Inspired by the world of *The Witcher*, the game allows you to explore mysterious lands, solve riddles, gather magical ingredients, and fight monstrous beasts.

## How to Play

The game can be launched in two modes:

### Graphical Mode

java -jar target/adv_witcher_labm02.jar

java -jar target/adv_witcher_labm02.jar text


### Available commands 

 - go [location]            Travel to a different location.
 - look                     Observe your surroundings.
 - inspect [item]           Examine item in detail.
 - inventory                View your inventory.
 - take [item]              Pick up an item.
 - drop [item]              Drop an item.
 - use [item]               Use an item.
 - combine [item1] [item2]  Craft or merge items.
 - attack [enemy]           Engage in combat.
 - cast [spell]             Use a Witcher sign.
 - solve_riddle [answer]    Solve a puzzle or riddle.
 - light [item]             Light torch or lantern.
 - talk to [character]      Interact with NPC.
 - give [item] [character]  Give an item to an NPC.
 - read [book/note]         Read item content.
 - unlock [object]          Use a key to unlock.
 - meditate                 Heal and restore.
 - help                     View help.
 - end / quit               End the game.
